About

PaySafe is an intelligent UPI fraud detection platform designed to enhance the security of digital transactions in real time. It leverages machine learning, anomaly detection, and explainable AI to detect suspicious behavior and protect users from financial fraud.

Overview

PaySafe is a real-time fraud detection system built to secure digital payment ecosystems such as UPI. The platform integrates advanced machine learning models, rule-based risk scoring, behavioral analytics, and anomaly detection to evaluate every transaction dynamically. Each transaction is analyzed across multiple dimensions including transaction amount, user behavior patterns, device identity, and contextual signals to generate an accurate risk score. The system classifies transactions into safe, suspicious, or high-risk categories and provides clear, explainable insights into why a transaction was flagged. With a FastAPI backend and an interactive frontend dashboard, PaySafe enables instant fraud detection, continuous monitoring, and intelligent decision-making for enhanced financial security.

How It Works

User inputs transaction data → Backend processes it → ML model predicts fraud probability → Anomaly model checks unusual behavior → Final risk score is generated → Result is displayed instantly with visual indicators.
